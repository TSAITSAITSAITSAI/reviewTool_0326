出發前總待辦

先把文件、票券、網路、交通與冬季裝備確認好，旅途中會輕很多。

必做清單

  • 確認澳洲 ETA / 簽證。
  • 確認旅遊保險，保單存在手機與雲端。
  • 準備護照影本與重要文件電子備份。
  • 確認國泰與 Virgin Australia 航班、訂位代號、行李額度。
  • 訂澳洲 eSIM / 漫遊。
  • 下載或收藏 Google Maps：Melbourne、Great Ocean Road、Phillip Island、Sydney、Blue Mountains。
  • 準備信用卡與少量澳幣現金。
  • 準備防風外套、輕便雨具、好走鞋。

建議先訂

  • 6/15 墨爾本晚餐：Tipo 00 / Chin Chin / Supernormal / Cumulus Inc.
  • 6/16 大洋路一日遊 KKday。
  • 6/17 Phillip Island 企鵝＋動物園 KKday。
  • 6/18 墨爾本 MEL → 雪梨 SYD 機票。
  • 6/19 藍山一日遊 KKday。
  • 6/20 若選 Taronga Zoo，先買門票。

6/13（六）

Day 0｜台北出發，香港轉機

TPE 12:00 → HKG 14:05，HKG 19:00 → MEL 05:55+1。

今日行程

  1. 台北桃園機場出發。
  2. 香港轉機 4h55m。
  3. 夜飛墨爾本，機上休息。

注意事項

  • 出門前確認護照、信用卡、手機、行動電源。
  • 香港轉機先確認下一段登機口。
  • 機上以睡眠、補水為主。

備用方案

  • 第一段延誤時先看轉機櫃台與登機口。
  • 機上睡不好，6/14 只保留輕鬆散步。
6/14（日）

Day 1｜抵達墨爾本，飯店，維多利亞市場

清晨抵達，今天主軸是恢復體力和熟悉飯店周邊。

今日行程

  1. 05:55 抵達 Melbourne Airport。
  2. Aura on Flinders 寄放行李。
  3. Federation Square / Flinders Street Station。
  4. Yarra River 南岸散步。
  5. Queen Victoria Market 可選。

餐廳安排

  • 早餐／早午餐：Higher Ground。
  • 午餐：Queen Victoria Market 或 Flinders Lane。
  • 晚餐：Cumulus Inc. 或 Chin Chin；太累就簡餐。

注意事項

  • 清晨抵達，不要急著跑行程。
  • Queen Victoria Market 週日可排，週一不營業。
  • 若無法提早入住，找咖啡廳坐下休息。
機場到飯店：搭 SkyBus 到 Southern Cross Station，再步行或短程 Uber / Taxi 到 Aura on Flinders。行李多或太累時直接 Uber / Taxi。
開啟：Melbourne Airport → Aura on Flinders 開啟：飯店 → Queen Victoria Market
6/15（一）

Day 2｜墨爾本 CBD，晚餐訂位

咖啡巷弄、街頭藝術、圖書館、購物和 Flinders Lane 晚餐。

地理順序

  1. Degraves Street / Centre Place。
  2. Hosier Lane。
  3. State Library Victoria。
  4. Melbourne Central / Emporium。
  5. Chinatown / Flinders Lane。
開啟 CBD 路線

晚餐訂位

  • Tipo 00：義大利麵，熱門。
  • Chin Chin：熱鬧泰式。
  • Supernormal：亞洲風味。
  • Cumulus Inc.：Flinders Lane，動線順。

注意與備案

  • Queen Victoria Market 週一不營業。
  • 墨爾本天氣變化快，帶外套和雨具。
  • 雨天：State Library → Melbourne Central → Emporium。
  • 低體力：刪 Hosier Lane。
6/16（二）

Day 3｜大洋路一日，訂 KKday

長途海岸線日：十二使徒岩、Loch Ard Gorge、London Bridge。

今日行程

  1. 墨爾本市區出發。
  2. Memorial Arch。
  3. Apollo Bay 午餐。
  4. Twelve Apostles。
  5. Loch Ard Gorge。
  6. London Bridge。
  7. 回墨爾本。

KKday 訂票

  • 確認是否含飯店接送或集合地點。
  • 確認集合時間，通常很早。
  • 確認是否含午餐。
  • 確認取消規定、天氣取消規則。

注意與備案

  • 前一晚早睡。
  • 帶防風外套、雨具、行動電源。
  • 容易暈車的人準備暈車藥。
  • 團取消：改 NGV / State Library / Emporium。
開啟大洋路參考路線
6/17（三）

Day 4｜Phillip Island 企鵝，訂 KKday

動物園、Cowes 提早晚餐、Penguin Parade。晚上戶外很冷，保暖優先。

今日行程

  1. 墨爾本出發。
  2. Moonlit Sanctuary 或 Maru。
  3. Cowes 提早晚餐。
  4. Nobbies Centre 可選。
  5. Phillip Island Penguin Parade。
  6. 回墨爾本。

KKday 訂票

  • 確認是否含動物園。
  • 確認集合時間、集合點、回程時間。
  • 確認座位：General / Penguins Plus / Underground。
  • 確認是否含晚餐。

注意與備案

  • 企鵝歸巢是晚上戶外，非常冷。
  • 通常不能拍照，也不要開手機螢幕。
  • 時間不足：刪 Nobbies Centre。
  • 下雨或風大：防風、防水、保暖。
開啟 Phillip Island 路線
6/18（四）

Day 5｜墨爾本到雪梨，機場交通與機票

MEL 11:00 → SYD 12:30；下午只排雪梨港區核心。

今日行程

  1. 墨爾本飯店退房。
  2. 前往 Melbourne Airport。
  3. 飛雪梨。
  4. Meriton Suites World Tower 放行李。
  5. Circular Quay、Opera House、The Rocks。

機票提醒

  • 訂 Virgin Australia MEL → SYD。
  • 確認行李額度。
  • 確認航廈與報到截止時間。
  • 前一天完成線上 check-in。

注意與備案

  • 早上不要排 Queen Victoria Market。
  • 退房前檢查護照、充電器、保險箱。
  • 航班延誤：只走港區核心。
  • 行李太多：機場到飯店直接 Uber / Taxi。
開啟：墨爾本飯店 → Melbourne Airport 開啟：Sydney Airport → 雪梨飯店 開啟：雪梨港區步行路線
6/19（五）

Day 6｜藍山一日，訂 KKday

Three Sisters、Scenic World、Leura / Katoomba。山區比市區冷。

今日行程

  1. 雪梨飯店出發。
  2. Central Station。
  3. Katoomba Station。
  4. Echo Point / Three Sisters。
  5. Scenic World。
  6. Leura 或 Katoomba。
  7. 回雪梨。

KKday 訂票

  • 確認是否包含 Scenic World 票券。
  • 確認是否去 Three Sisters、Echo Point、Leura。
  • 確認集合時間、集合點、回程時間。
  • 不跟團需查火車與當地交通。

注意與備案

  • 帶外套、好走鞋、水、零食。
  • 山區可能起霧，等天氣變化。
  • 體力不足：只做 Echo Point + Scenic World。
  • 天氣很差：改室內雪梨日。
開啟藍山公共交通路線
6/20（六）

Day 7｜Circular Quay 路線圖，Taronga Zoo 或 Manly Beach

以 Circular Quay 為核心，依天氣和體力二選一。

A 方案：Taronga Zoo

  1. 飯店 → Circular Quay。
  2. Circular Quay Wharf 搭船到 Taronga Zoo。
  3. Taronga Zoo 參觀。
  4. 回港區，接 Barangaroo / Darling Harbour 晚餐。
  • 先買門票。
  • 查當天渡輪時刻。
  • 園區有坡度，穿好走鞋。
開啟 Taronga Zoo 路線

B 方案：Manly Beach

  1. 飯店 → Circular Quay。
  2. Circular Quay 搭 Manly Ferry。
  3. Manly Wharf → The Corso → Manly Beach。
  4. 體力夠再走 Shelly Beach。
  • Manly Ferry 是很好的港景路線。
  • 海邊風大，帶外套。
  • 假日人多，回程不要抓太晚。
開啟 Manly Beach 路線
雨天備案：Australian Museum → QVB → Westfield Sydney。低體力版只去 Circular Quay、Opera House 外觀、The Rocks。
6/21（日）

Day 8｜回程日，最後購物與機場

完全不趕：寄放行李、最後購物、18:30-19:00 前往機場。

今日行程

  1. Meriton 退房 / 寄放行李。
  2. QVB。
  3. Westfield Sydney / Pitt Street Mall。
  4. Hyde Park / St Mary's Cathedral 可選。
  5. 回飯店拿行李。
  6. Sydney Airport。

注意事項

  • 17:00 前吃完早晚餐。
  • 回飯店拿行李前預留至少 30 分鐘。
  • 有退稅需求要提早到機場。
  • 檢查押金、證件、充電器。

備用方案

  • 下雨：QVB → Westfield → Pitt Street Mall。
  • 行李多：全程寄放飯店。
  • 交通延誤：直接 Uber / Taxi 到機場。
  • 更保守：18:00 出發。
開啟飯店 → Sydney Airport

額外注意事項

交通付款、訂位策略、天氣策略與每晚睡前檢查。

交通付款

  • 墨爾本：Free Tram Zone 內搭電車不用刷 myki；出 Free Tram Zone 才需要付費。
  • 雪梨：可用 Opal 或支援的信用卡 / 行動支付，記得 tap on / tap off。

訂位策略

  • 熱門餐廳不要排在大洋路、企鵝、藍山這三個長途日晚上。
  • 長途日晚上以近飯店、可取消、簡單吃為主。
  • 需要早起的一日遊，前一晚不要訂太晚。

每晚睡前確認

  • 隔天集合點與集合時間。
  • 門票 / KKday 訂單。
  • 天氣與 Google Maps 路線。
  • 行動電源充電。
  • 護照、錢包、手機位置。
